# npm-scripts static template

## Installation
```bash
$ npm i
$ npm install -g backstopjs
$ backstop init
```

## Usage
```bash

# rm -rf dist
$ npm run clean

# build js
$ npm run build:js

# build ejs
$ npm run build:ejs

# build sass
$ npm run build:sass

# Minify images
$ npm run build:images

# build src files
$ npm run build

# format sass
$ npm run format

# develop server
$ npm run serve

# dev
$ npm run dev

# create reference files
$ reference:visual

# visual regression testing
$ test:visual

# update reference files
$ approve:visual
```
